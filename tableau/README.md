# Tableau Dashboards

Analytics dashboards built from the star schema.

## Dashboards

### 1. Customer Satisfaction Dashboard
- Average satisfaction score by funeral home
- Complaint rate trending
- Family feedback analysis
- Response time to customer requests

### 2. Service Performance Dashboard
- On-time delivery % by region
- Service issues breakdown
- SLA compliance metrics
- Cremation/burial completion times

### 3. KPI Summary Dashboard
- Executive overview (5-6 KPIs on one page)
- Customer satisfaction trend
- Complaint rate gauge
- On-time delivery gauge
- Funerals handled metric
- Active policies count

## How to Use
1. Import CSV files from `data/sample-data/`
2. Connect dimensions and facts
3. Create calculated fields for KPIs
4. Build visualizations
5. Combine into dashboards

## Expected Results
- Fast query performance (pre-calculated metrics)
- Clean dimensional joins
- No data duplication
- Accurate KPI calculations
